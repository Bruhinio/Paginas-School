# 🏫 Colegio Luminis — Sitio Web Primaria

Sitio web oficial de la sección de Primaria del **Colegio Luminis**, con panel de configuración integrado.

## 🚀 Ver el sitio en línea

👉 [https://TU-USUARIO.github.io/TU-REPOSITORIO](https://TU-USUARIO.github.io/TU-REPOSITORIO)

> Reemplaza `TU-USUARIO` y `TU-REPOSITORIO` con tu usuario y nombre de repositorio de GitHub.

## 📁 Estructura del proyecto

```
/
├── index.html       ← Sitio completo con panel de configuración

```

---

## ⚙️ Panel de configuración

El sitio incluye un **panel de configuración** accesible desde el botón ⚙️ en la esquina inferior derecha. Desde ahí puedes cambiar:

- 🏫 Nombre, siglas y logo del colegio
- 🎨 Fondo del hero (degradado de colores o imagen propia)
- 📷 Foto de la sección Misión y galería de instalaciones (hasta 6 fotos)
- 📊 Estadísticas (números y etiquetas)
- 📞 Teléfonos, direcciones y horarios de contacto

---

## 🌐 Cómo publicar en GitHub Pages

1. Sube los archivos a tu repositorio de GitHub
2. Ve a **Settings → Pages**
3. En **Source**, selecciona la rama `main` y la carpeta `/ (root)`
4. Haz clic en **Save**
5. En unos segundos tendrás tu URL pública
